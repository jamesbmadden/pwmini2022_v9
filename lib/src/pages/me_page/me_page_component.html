Naming System

Version.Release.Patch
Version aligns with the current grade, and represents a major, breaking redesign or update, and usually comes with significant design updates.
Releases are monthly updates containing both new features and bugfixes, and slight design changes. Any homework posted in any release of a version should alwaus be compatible with any other release of that version.
Patches are the bugfix updates, coming out whenever neccesary, usually soon after a release update or version update. Patches will almost never include design changes or new features, unless the old design was caused by a bug or feature needed to fix bug.
